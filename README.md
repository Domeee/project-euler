# Don Hubi's take on the [Project Euler](https://projecteuler.net)

## Programming languages

### TypeScript

Install prerequisites

```bash
cd typescript
npm install
 ```
Development mode
 ```bash
tsc --watch
npm start
 ```

Run EulerTest.ts (with vscode debugger) to get the results
